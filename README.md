# MeadowQuest

MeadowQuest is a small 3D browser game set in a cozy fantasy meadow village. You control a tiny traveller, explore round-door cottages and rolling hills, and collect glowing herbs around the map.

## Features

- 3D world rendered with Three.js
- Third-person controllable character
- WASD / arrow-key movement
- Q / E camera rotation
- Cottages, trees, paths, hills, lighting, shadows, and fog
- Collectible glowing herbs with a live score counter
- Responsive full-screen game UI

## How to run

Because the game uses JavaScript modules, run it through a local server instead of opening the file directly.

If you have Python installed:

```bash
python -m http.server 5500
```

Then open:

```text
http://localhost:5500
```

## Controls

- `WASD` or arrow keys: move
- `Q` / `E`: rotate camera
- Collect all glowing herbs to complete the run
